toolbox
================

本人在go语言开发中经常使用的工具包。

## 1 common

基于go语言基础库，经常使用到的函数。

* 错误处理
* 文件处理
* http rest 客户端
* json处理
* 系统环境处理
* struct tag标签处理

引用
```golang
import github.com/lordking/common
```
